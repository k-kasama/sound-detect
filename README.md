# sound-detect

## 目的
- 雑音含め鳴った音が、既知の音源かどうか判別するプロダクトを作成する

## architecture
on docker  
django + pyinstallerで頒布

  - djangoでロケット発射

  - 周波数表示欄を設置

- 単音(単周波数)をききわける
  - 単音を作成する
